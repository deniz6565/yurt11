# Symfony Projesi

Bu proje, Symfony framework'ü kullanılarak geliştirilmiştir. Projenin temel amacı şu işlevselliği sağlamaktır:yurt otomasyonu katır işlemleri silme güncelleme oda işlemler bu projede yapılması
amaçlanmıştır.

## Başlangıç

localhostta calısıyor bunun için sunucu olarak xampp ı kurdum ve symfony server:start komutu ile çalıştırdım.

### Gereksinimler

Projeyi çalıştırmak için gerekli yazılım ve kütüphaneler:

- PHP 8.3+
- Composer
- Symfony CLI (Opsiyonel)
- Doctrine
  
composer install
Kurulum
Projeyi yerel makinenizde çalıştırmak için adımlar:

Depoyu klonlayın:
git clone https://github.com/kullanici/proje.git

Proje dizinine gidin:
cd proje
composer install
web sunucusu başlatılır symfony server:start
